# ashkara_tbilisi_bot
Telegram-бот для приёма и учёта списаний на производстве.  
Позволяет сотрудникам отправлять сообщения в установленном формате, автоматически разбирает их и сохраняет в базу данных PostgreSQL.

## Возможности
- Обрабатывает списания по категориям: порча, стафф и прочее
- Сохраняет данные в PostgreSQL
- Встроенная авторизация по коду

## Технологии
- `Python` 3.11+
- `Aiogram` 3+
- `PostgreSQL` + `SQLAlchemy`
- `asyncpg`, `alembic`
